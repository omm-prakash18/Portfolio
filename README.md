## ✨ Key Features

- **Interactive WebGL Background:** A custom "blueprint" shader grid that animates in the background, complete with light/dark mode color shifting.
- **Light & Dark Theme Toggle:** Seamless theme switching utilizing CSS custom properties and `localStorage` for state persistence.
- **3D Tilt Effects:** Smooth, perspective-driven 3D hover effects on project and skill cards for fine-pointer devices.
- **Performance & Accessibility Optimized:** Includes `prefers-reduced-motion` detection to pause WebGL animations for users who experience motion sickness, and optimized font loading.
- **Responsive Design:** Fluid layouts that adapt beautifully across mobile, tablet, and desktop screens.
- **Working Contact Form:** Integrated with Formspree for easy, serverless message delivery.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla, CSS Variables, Flexbox/Grid)
- **Scripting:** Vanilla JavaScript (ES6+), DOM Manipulation, Intersection Observer API
- **Graphics:** WebGL (for background shaders)
- **Icons & Typography:** FontAwesome 6, Google Fonts (Inter, Fira Code)
